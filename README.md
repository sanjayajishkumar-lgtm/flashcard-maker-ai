# AI-Powered Personal Study Flashcard Maker

This project was developed as part of a 15-day internship on Artificial Intelligence
and Generative AI under YBI Foundation.

## Features
- Automatically generates flashcards from study notes
- Uses NLP and Generative AI
- Simple and beginner-friendly
- Exportable flashcards

## Technologies Used
- Python
- Google Colab
- HuggingFace Transformers
- NLP

## How to Run
1. Open the notebook in Google Colab
2. Install required libraries
3. Enter your study text
4. Generate flashcards instantly

- Organization: YBI Foundation
